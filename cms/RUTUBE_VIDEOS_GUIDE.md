# Управление видео VK через CMS

## Описание

Этот модуль позволяет управлять видео из ВКонтакте, которые отображаются на главной странице и странице маркетинговой активности.

## Как добавить или изменить видео

### 1. Войдите в админ-панель Strapi

Перейдите по адресу: `http://localhost:1337/admin` (или ваш production URL)

### 2. Перейдите в раздел "Видео Rutube"

В левом меню найдите пункт **"Видео Rutube"**

### 3. Добавление нового видео

1. Нажмите кнопку **"Create new entry"**
2. Заполните поля:
   - **Title** (обязательно) - название видео (для внутреннего использования)
   - **Video Url** (обязательно) - полная ссылка на видео из ВКонтакте
   - **Description** (опционально) - описание видео
   - **Order** - порядок отображения (0 = первое, 1 = второе, и т.д.)
   - **Show On Home Page** - показывать ли видео на главной странице
   - **Show On Marketing Page** - показывать ли видео на странице маркетинговой активности

3. Нажмите **"Save"** и затем **"Publish"**

### Как добавить ссылку на видео ВКонтакте

**Просто скопируйте полный URL** из адресной строки браузера:
```
https://vk.com/video-74008963_456239392
```

Система автоматически преобразует ссылку в формат для встраивания на сайт.

### 4. Редактирование существующего видео

1. В списке видео кликните на запись, которую хотите отредактировать
2. Внесите изменения
3. Нажмите **"Save"** и **"Publish"**

### 5. Удаление видео

1. В списке видео кликните на запись
2. Нажмите кнопку **"Delete this entry"**
3. Подтвердите удаление

## Важные моменты

- **Видео можно показывать одновременно** на главной странице и на странице маркетинговой активности
- **Order** определяет последовательность отображения (меньшее число = раньше)
- Если в CMS нет опубликованных видео, будут показаны дефолтные видео из кода
- Изменения вступают в силу после публикации (Publish)
- На сайте видео обновляются автоматически при перезагрузке страницы

## Технические детали

- Content Type: `rutube-video` (Collection Type)
- API Endpoint: `/api/cms/rutube-videos`
- Компонент: `RutubeVideos`
- Используется на страницах:
  - Главная (`/`)
  - Маркетинговая активность (`/marketing-activity`)
